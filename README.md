# QuestStreet

QuestStreet — проект городских офлайн-квестов с сюжетным прохождением через Telegram-бота.  
Квесты рассчитаны на самостоятельное прохождение по городу с использованием смартфона.

________________________

## О проекте

Проект запущен в 2023 году без команды и внешних инвестиций.  
За время разработки было создано:
- 4 квеста в Москве  
- 3 квеста в Сочи  

Средняя продолжительность одного квеста — ~2 часа.  
Каждый квест включает ~15 точек по городу и сюжетную линию.

_________________________________________________

## Формат квестов

- Прохождение через Telegram-бота  
- Геолокационные точки в городе  
- Сюжетное повествование и задания  
- Самостоятельное прохождение без ведущего  

_________________________________________________________

## Техническая часть

- Telegram-бот
- Управление состояниями пользователя
- Интеграция оплаты (самозанятый / ИП)
- Админ-панель для контроля прохождения квестов
- Интегрирована БД с данными users для анализа и админ-панели
  
________________________________________________

## Монетизация

- Продажа доступа к квестам
- Партнёрские модели с площадками (парки, городские пространства)
- Тестировались модели разделения выручки (rev-share)
